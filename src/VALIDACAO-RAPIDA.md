# ✅ Validação Rápida - 2 Minutos

## 🎯 Checklist de Validação

### 1️⃣ Cenários (30 segundos)
```
1. Abrir "Cenários & Comparação"
2. Verificar se carrega lista de cenários
3. Clicar em "Aplicar" em algum cenário
4. Ver toast de sucesso ✅
```

### 2️⃣ Relatórios (45 segundos)
```
1. Abrir "Relatórios & KPIs"
2. Ver 4 cards de resumo no topo
3. Clicar na aba "Atendimento Demanda"
4. Verificar gráfico de barras ✅
5. Clicar na aba "Trocas de Módulo"
6. Verificar análise de impacto ✅
```

### 3️⃣ Configurações (45 segundos)
```
1. Abrir "Configurações"
2. Ver sliders de α, β, γ, δ
3. Arrastar um slider
4. Ver valor mudando ✅
5. Clicar em "Salvar Parâmetros Padrão"
6. Ver toast de sucesso ✅
7. Ir para aba "Supabase"
8. Verificar badge verde "Conectado" ✅
```

## ❌ Se Algo Falhar

### Problema: Página não carrega
**Solução**: Verificar console do navegador (F12)

### Problema: Erro ao salvar
**Solução**: Verificar conexão Supabase

### Problema: Gráfico vazio
**Solução**: Normal se não houver dados no período
